1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent.

//code fragment 1

$("li").each(function(idx, e) {
$(e).css(“color”, “yellow”); });

//code fragment 2

$("li").each(function() {
$(this).css(“color”, “yellow”); });

//code fragment 3

$("li").each(function(idx) {
$(this).css(“color”, “yellow”); });

[Answer]

fragment 1 will change all the element colors to yellow because it is looping over `e`
fragment 2 will change only element where event occured because of "this"
fragment 3 will change only element where event occured because of "this"
